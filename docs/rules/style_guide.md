# Style Guide: Minimalist Earth Tones

This style guide defines the visual design for our application. The aesthetic is minimalist and functional, with a color palette inspired by earth tones.

## Philosophy

- **Minimalism**: Every element should have a purpose. Avoid unnecessary decoration.
- **Functionality**: The design should be intuitive and easy to use.
- **Content-Focused**: The design should not distract from the content.

## Color Palette

The color palette consists of muted earth tones, creating a calm and focused user experience.

- **Primary:** `#3D2B1F` (Dark Brown) - Used for primary actions and headings.
- **Secondary:** `#8F7C6A` (Muted Brown) - Used for secondary actions and borders.
- **Accent:** `#5F7161` (Muted Green) - Used for highlights and selected states.
- **Background:** `#FDFBF7` (Off-White) - The primary background color.
- **Text:** `#2A2A2A` (Charcoal) - The primary text color.

## Typography

- **Font Family**: Use a sans-serif font stack for readability. `system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif`
- **Headings**: Use the primary color and a bold font weight.
- **Body Text**: Use the text color and a regular font weight.
- **Font Sizes**: Use a modular scale for font sizes.
  - H1: 2.488rem
  - H2: 2.074rem
  - H3: 1.728rem
  - H4: 1.44rem
  - P: 1.2rem
  - Small: 1rem


## Layout

- **Grid System**: Use a 12-column grid system for layout.
- **Spacing**: Use a consistent spacing scale based on 8px increments (8px, 16px, 24px, 32px, etc.).
- **White Space**: Use generous white space to create a clean and uncluttered layout.

## Components

### Buttons

- **Primary Button**: Solid background color (`#3D2B1F`) with light text.
- **Secondary Button**: Outlined with the secondary color (`#8F7C6A`) and dark text.
- **Hover/Active States**: Use the accent color (`#5F7161`) to indicate hover and active states.

### Forms

- **Inputs**: Simple, underlined inputs with a focus state that uses the accent color.
- **Labels**: Placed above the input field.

### Other Elements

- **Borders**: Use the secondary color (`#8F7C6A`) for borders.
- **Shadows**: Avoid shadows. Use borders to create separation between elements.

This style guide provides a foundation for a consistent and visually appealing user interface. All new components and features should adhere to these guidelines.
